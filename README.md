# Elysium
A C++ game engine built for 2D style game.

## Quick Start

### Linux/Mac

```bash
chmod +x ./elysium.sh

# Setup build environment for the first time
./elysium.sh --Setup

# Clean, build, and run
./elysium.sh --clean --build --run
```

### Windows (PowerShell)

```powershell
# Setup build environment for the first time
.\elysium.ps1 --Setup

# Clean, build, and run
.\elysium.ps1 --Clean --Build --Run
```

## Requirements

- CMake 3.20+
- C++17 compatible compiler
- OpenGL support (for Raylib)
